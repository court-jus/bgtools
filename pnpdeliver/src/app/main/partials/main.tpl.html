PNP Deliver

A tool to help game designers distribute their PnP games and to help PnP players to print them.

Et en français ça donne quoi ?

C'est un outil pour aider les auteurs de jeux à distribuer leurs jeux en PnP et pour aider les joueurs à imprimer ceux-ci.

Cool non ?

Carrément !

Last changes

  • 0.1.0
    • There's a light gray background to show where the page is
    • Automatic creation of a deck from a list of image URLs has been moved to the Deck section
    • Save and Load have been moved to a new "Game" section that allows to edit game name, author and description
    • Game information is shown on the preview and can be hidden by clicking the game name above the forms
  • 0.0.9
    • A background color can be added in the margins
    • A page break can be added between the decks
    • Two UI are now available (simple/advanced)
    • Code cleaned
  • 0.0.8
    • Crop marks now work on the back side.
    • Can rename a deck.
    • Can apply a global card count or card back to a deck.
    • Crop marks are centered on the inner margins.
  • 0.0.7
    • Crop marks can be added on the front side of the cards (back side not functionnal yet).
  • 0.0.6
    • Card back can be displayed, side by side with the front side or on a separate page for duplex printing.
  • 0.0.5
    • Can now save and load.
    • The saved JSON file can be shared on a public repository (dropbox) and can be passed through the URL like this : http://court-jus.github.io/PnPDeliver/#/advanced?load=MYPUBLICURL
  • 0.0.4
    • When display boundaries, the images are not hidden, but their opacity is reduced.
    • Cards overflow is hidden so the cropping effect of having images bigger than cards is better.
    • Options are organized in categories.
    • Multiple copies of the same card can be created.
    • Multiple decks can be generated and managed.
  • 0.0.3
    • It is now possible to use images with bleeding by specifying image width/height and image shift. For example if you are making standard 2.5/3.5 cards and have a 1/10in bleed, you will choose card width=2.5, card height=3.5, image width=2.7, image height=3.7, image horizontal shift = -0.1 and image vertical shift = -0.1.
  • 0.0.2
    • It is now possible to choose the number of cards per page (rows, cols), cards size and the cards margin.
