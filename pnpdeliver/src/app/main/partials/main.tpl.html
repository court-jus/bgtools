PNP Deliver

A tool to help game designers distribute their PnP games and to help PnP players to print them.

Et en français ça donne quoi ?

C'est un outil pour aider les auteurs de jeux à distribuer leurs jeux en PnP et pour aider les joueurs à imprimer ceux-ci.

Last changes

  • 0.0.5
    • Can now save and load.
    • The saved JSON file can be shared on a public repository (dropbox) and can be passed through the URL like this : http://court-jus.github.io/PnPDeliver/#/test?load=MYPUBLICURL
  • 0.0.4
    • When display boundaries, the images are not hidden, but their opacity is reduced.
    • Cards overflow is hidden so the cropping effect of having images bigger than cards is better.
    • Options are organized in categories.
    • Multiple copies of the same card can be created.
    • Multiple decks can be generated and managed.
  • 0.0.3
    • It is now possible to use images with bleeding by specifying image width/height and image shift. For example if you are making standard 2.5/3.5 cards and have a 1/10in bleed, you will choose card width=2.5, card height=3.5, image width=2.7, image height=3.7, image horizontal shift = -0.1 and image vertical shift = -0.1.
  • 0.0.2
    • It is now possible to choose the number of cards per page (rows, cols), cards size and the cards margin.
